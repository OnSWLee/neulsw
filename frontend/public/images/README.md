# 이미지 폴더 구조

이 폴더는 홈페이지에 사용할 이미지들을 저장하는 곳입니다.

## 폴더 구조

- `home/` - 메인 페이지용 사진
- `doctor/` - 원장 약력용 사진
- `clinics/` - 클리닉용 사진
- `reviews/` - 후기용 사진

## 사용 방법

각 폴더에 이미지를 업로드한 후, 코드에서 다음과 같이 참조할 수 있습니다:

```tsx
// 예시: 메인 페이지에서
<img src="/images/home/main-image.jpg" alt="메인 이미지" />

// 예시: 원장 약력 페이지에서
<img src="/images/doctor/doctor-photo.jpg" alt="원장 사진" />
```

## 참고사항

- Vite에서는 `public` 폴더의 파일들이 루트 경로(`/`)에서 제공됩니다.
- 이미지 파일명은 영문과 숫자, 하이픈(-), 언더스코어(_)를 사용하는 것을 권장합니다.
- 권장 이미지 형식: JPG, PNG, WebP
